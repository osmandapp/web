# Routes {#routes}

OsmAnd has many powerful features for displaying different [routes](./types-of-routes.md) on the map. By default, these routes are part of [vector maps](../../map/vector-maps.md#routes) (OpenStreetMap data), however similar functionality is provided by [tracks](../../map/tracks/index.md) which could be created with [Plan a Route](../../plan-route/create-route.md), imported as [GPX tracks](./routes-on-map.md#save-as-a-track), recorded with the [Trip Recording plugin](../../plugins/trip-recording.md). Routes Map Ledend is [here](../../map-legend/osmand.md#routes).
