# Web 1.03 {#web-103}

Web 1.03 ist die neueste Webversion von OsmAnd, die neue nützliche Funktionen und Verbesserungen enthält. Dieser Artikel beschreibt alle wichtigen Updates des [Web-Planers](../user/web/index.md).


## Was ist neu {#whats-new}

- Neugestaltung der Benutzeroberfläche der Navigation
- Unterstützung für smarte Ordner
- Einstellungen für das Erscheinungsbild von Tracks (Farbe & Breite)
- Haltestellen des öffentlichen Verkehrs
- Verbesserungen am Kontextmenü für POIs


- [Optionale Updates](#optional-updates). Zusätzliche Updates und Verbesserungen für die Version.


## Optionale Updates {#optional-updates}

- [GPX als vereinfachten Track exportieren](https://github.com/osmandapp/web/issues/436)
- [Öffnungszeiten im Kontextmenü für POIs angezeigt](https://github.com/osmandapp/web/issues/1277)
- [Festgepinnte Favoritenordner](https://github.com/osmandapp/web/issues/1193)
- [Verbesserungen bei Wikimedia-Metadaten](https://github.com/osmandapp/OsmAnd-Issues/issues/2959)
- [Verbessertes Verhalten des Ebenenmenüs](https://github.com/osmandapp/web/issues/918)
- [Track-Tooltip beim Hovern](https://github.com/osmandapp/web/issues/1305)
- [Suchverbesserungen für persönliche Daten](https://github.com/osmandapp/web/issues/1065)
- [Verbesserungen der POI-Galerie](https://github.com/osmandapp/OsmAnd/issues/22671)
- [Verbesserungen bei der Markerauswahl](https://github.com/osmandapp/web/issues/1297)
- [Verbessertes Suchverhalten für POI-Kategorien](https://github.com/osmandapp/web/issues/1126)
- [Abgelaufene Käufe auf der Konto-Seite ausblenden](https://github.com/osmandapp/web/issues/1328)
