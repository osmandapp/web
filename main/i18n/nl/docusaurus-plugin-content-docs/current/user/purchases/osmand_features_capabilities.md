## Alle functies en mogelijkheden {#all-features-and-capabilities}

OsmAnd is **een krachtige navigatie- en kaartenapp** die een breed scala aan offline en online functies biedt. Deze pagina geeft een gedetailleerd overzicht van alle beschikbare functies in verschillende versies van de app.



## Navigatie {#navigation}

Kernnavigatiefuncties:

- Turn-by-turn navigatie met gesproken aanwijzingen
- Automatische herberekening bij afwijking van de route
- Geschatte aankomsttijd (ETA)
- Ondersteuning voor tussenliggende waypoints
- Rijstrookbegeleiding
- Waarschuwingen voor snelheidslimieten
- Verkeersbewuste routeaanpassingen

Ondersteunde vervoerswijzen:

- Auto
- Fiets
- Voetganger
- Openbaar vervoer
- Maritieme navigatie
- Wandelen

Tabel:

| Functie | OsmAnd Gratis | OsmAnd+ (Kaarten+) | OsmAnd Pro |
|---------|------------|----------------|------------|
| Turn-by-turn navigatie | ✔ | ✔ | ✔ |
| Rijstrookbegeleiding | ✔ | ✔ | ✔ |
| Tussenliggende waypoints | ✔ | ✔ | ✔ |
| Verkeersbewuste herberekening | - | - | ✔ |
| Android Auto / CarPlay | - | ✔ | ✔ |


## Kaarten en weergave {#maps-and-rendering}

Offline kaarten en updates:

- Downloadbare regionale en landkaarten
- Maandelijkse kaartupdates
- Uurlijkse updates via OsmAnd Live

Ondersteunde kaarttypen:

- Standaard OpenStreetMap kaart
- Satellietkaarten (online)
- 3D Reliëfkaarten
- Contourlijnen (Topografie)
- Nautische dieptes en hydrologie

Extra lagen:

- Wegdek en kwaliteit
- Wandelpaden en fietsroutes
- Offline OpenStreetMap lagen
- Dynamische weerlagen

Tabel:

| Functie | OsmAnd Gratis | OsmAnd+ (Kaarten+) | OsmAnd Pro |

| Offline kaarten | ✔ | ✔ | ✔ |

| Uurlijkse updates (OsmAnd Live) | - | - | ✔ |
| 3D Reliëf | - | - | ✔ |
