# Web 1.02 {#web-102}

Web 1.02 is de nieuwste webversie van OsmAnd, met nieuwe handige functies en verbeteringen. Dit artikel beschrijft alle belangrijke updates van de [Web Routeplanner](../user/web/index.md).


## Wat is er nieuw {#whats-new}

- Deel bestanden GPX / favorieten
- Navigatie Redesign website
- Klik op de kaart (vector tegels)
- Segmentanalyse


- [Optionele updates](#optional-updates). Aanvullende updates en verfijningen voor de release.
