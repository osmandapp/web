# Web 1.03 {#web-103}

Web 1.03 is de nieuwste webversie van OsmAnd, die nieuwe nuttige functies en verbeteringen bevat. Dit artikel beschrijft alle belangrijke updates van de [Webplanner](../user/web/index.md).


## Wat is er nieuw {#whats-new}

- Herontwerp van de navigatie-UI
- Ondersteuning voor slimme mappen
- Trackweergave-instellingen (kleur & breedte)
- Openbaarvervoerhaltes
- Verbeteringen aan het contextmenu van POI's


- [Optionele updates](#optional-updates). Aanvullende updates en verfijningen voor de release.


## Optionele updates {#optional-updates}

- [GPX exporteren als vereenvoudigde track](https://github.com/osmandapp/web/issues/436)
- [Openingstijden weergeven in het contextmenu van POI](https://github.com/osmandapp/web/issues/1277)
- [Vastgezette favorietenmappen](https://github.com/osmandapp/web/issues/1193)
- [Verbeteringen aan Wikimedia-metadata](https://github.com/osmandapp/OsmAnd-Issues/issues/2959)
- [Verbeterd gedrag van het laagmenu](https://github.com/osmandapp/web/issues/918)
- [Track-tooltip bij hover](https://github.com/osmandapp/web/issues/1305)
- [Zoekverbeteringen voor persoonlijke gegevens](https://github.com/osmandapp/web/issues/1065)
- [Verbeteringen aan de POI-galerij](https://github.com/osmandapp/OsmAnd/issues/22671)
- [Verbeteringen aan markeringselectie](https://github.com/osmandapp/web/issues/1297)
- [Verbeterd zoekgedrag voor POI-categorieën](https://github.com/osmandapp/web/issues/1126)
- [Vervallen aankopen verbergen op de accountpagina](https://github.com/osmandapp/web/issues/1328)
