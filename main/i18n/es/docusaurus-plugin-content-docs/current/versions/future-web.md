# Web 1.03 {#web-103}

Web 1.03 es la última versión web de OsmAnd, que incluye nuevas características y mejoras útiles. Este artículo describe todas las actualizaciones principales del [Planificador Web](../user/web/index.md).


## Novedades {#whats-new}

- Rediseño de la interfaz de usuario de navegación
- Soporte para carpetas inteligentes
- Configuraciones de apariencia de pista (color y ancho)
- Paradas de transporte público
- Mejoras en el menú contextual de POI


- [Actualizaciones opcionales](#optional-updates). Actualizaciones y refinamientos adicionales para el lanzamiento.


## Actualizaciones opcionales {#optional-updates}

- [Exportar GPX como pista simplificada](https://github.com/osmandapp/web/issues/436)
- [Horarios de apertura mostrados en el menú contextual de POI](https://github.com/osmandapp/web/issues/1277)
- [Carpetas de favoritos fijadas](https://github.com/osmandapp/web/issues/1193)
- [Mejoras en los metadatos de Wikimedia](https://github.com/osmandapp/OsmAnd-Issues/issues/2959)
- [Comportamiento mejorado del menú de capas](https://github.com/osmandapp/web/issues/918)
- [Información emergente de pista al pasar el cursor](https://github.com/osmandapp/web/issues/1305)
- [Mejoras en la búsqueda de datos personales](https://github.com/osmandapp/web/issues/1065)
- [Mejoras en la galería de POI](https://github.com/osmandapp/OsmAnd/issues/22671)
- [Mejoras en la selección de marcadores](https://github.com/osmandapp/web/issues/1297)
- [Comportamiento mejorado de la búsqueda de categorías de POI](https://github.com/osmandapp/web/issues/1126)
- [Ocultar compras caducadas en la página de cuenta](https://github.com/osmandapp/web/issues/1328)
