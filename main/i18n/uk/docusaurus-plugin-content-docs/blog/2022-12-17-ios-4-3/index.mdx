## New purchase plans

In this release, we added [new subscription plans](https://osmand.net/docs/user/purchases/ios) named _"OsmAnd Pro"_ and _"OsmAnd Maps+"_. OsmAnd Pro is a new subscription plan with access to all features, and "Maps+" - with unlimited map downloads and access to paid plugins.

:::note
_OsmAnd Pro_ is a cross-platform subscription (you can use it for both Android and iOS versions of OsmAnd after registration on our servers).
:::

Read more about iOS purchases [here](https://osmand.net/docs/user/purchases/ios).

### Weather forecast

Hourly forecast for the current day and 7 days ahead on the map.

[The Weather plugin](https://osmand.net/docs/user/plugins/weather): _Menu → Plugins → Weather_

Download the weather forecast in the _Maps & Resources_ menu for offline use:  _Menu → Maps & Resources → Map Region → Weather forecast_.

Weather widgets: _Temperature, Pressure, Wind, Cloud, Precipitation_. Shows these parameters for the central point on a device screen.

Weather screen menu: _Menu → Weather_

Weather map layers: _Menu → Configure map → Weather_

## New updates for GPX files

Support of [custom extensions](https://osmand.net/docs/technical/osmand-file-formats/osmand-gpx), link, cmt, desc tags.

1. Favorites: show content of ```<cmt>``` tag similar to waypoints:

```
<wpt lat="30.297375" lon="-87.4320327">
  <name>Caf� Beignet</name>
  <cmt>Open 6:30 am to 12:00 pm. Closed Sun &amp; Wed</cmt>
</wpt>
```

2. Phone numbers for the Waypoint attributes:

```
<wpt lat="52" lon="5">
<extension>
<phone>My phone number</phone>
<website>https://test.website</website>
<comment>Comment</comment>
<extension>
</wpt>
```

3. Tags for description: name, desc, cmt, link.

## FPS widget

FPS widget is the part of [OsmAnd development plugin](https://osmand.net/docs/user/plugins/development).

This is a [Development plugin](https://osmand.net/docs/user/plugins/development) widget to investigate the speed of displaying and updating the map and map elements. It displays:

[Map refresh FPS](https://osmand.net/docs/user/widgets/info-widgets#-development-widgets) - update points and routes on the map (smaller number).

You need to switch on _Show MTB IMBA trails_ in: _Menu → Configure map → Routes → Mountain bike → MTB Scale/IMBA_.
