## Огляд {#overview}

**Tracks Analyzer** — це веб-інструмент, який допомагає аналізувати повторювані сегменти треків між вибраними точками на карті. Він сканує ваші треки та знаходить усі сегменти, що проходять через обране(і) місце(а), дозволяючи вам порівнювати швидкість, висоту, відстань і час у кількох активностях.

## Як використовувати {#how-to-use}

Після відкриття Аналізатора треків (позначеного як гайковий ключ) інструмент відкривається з видом карти та порожнім станом. Звідси ви можете вибрати, які треки будуть включені до аналізу, за допомогою панелі **Вибір треків**. Аналізатор дозволяє працювати з усіма доступними треками або обмежити аналіз конкретними папками.

Щоб розпочати аналіз, встановіть одну або дві точки безпосередньо на карті. Клацніть правою кнопкою миші на бажаному місці та виберіть **Точка A / Точка B** з контекстного меню. Аналізатор тоді шукає сегменти треків, що проходять через вибрану точку або між двома точками.

## Сортування та видимі параметри {#sorting-and-visible-parameters}
Після того, як аналізатор знайде відповідні сегменти, результати відображаються у вигляді списку. Список можна переупорядкувати за допомогою опції **Сортування**, яка змінює спосіб розміщення сегментів. Крім того, кнопка **Поля** відкриває панель видимих параметрів, де ви можете керувати тим, які параметри аналізу відображаються для кожного сегмента. Ви можете відобразити всі доступні параметри або вибрати лише ті, що стосуються вашого аналізу.

Доступні параметри згруповані за типом:

**Швидкість**
- Макс. швидкість
- Середня швидкість
- Мін. швидкість

**Висота**
- Макс. висота
- Середня висота
- Мін. висота

**Підйом / Спуск**

**Дата та час**
- Дата
- Час початку
- Час завершення
- Часовий інтервал
- Тривалість
- Час у русі

**Довжина**

## Аналіз даних {#data-analysis}

Кожен відповідний сегмент відображається у списку результатів зліва. Для кожного сегмента відображається набір обчислених параметрів залежно від увімкнених видимих параметрів.

Кожен сегмент також має меню з трьома крапками (⋮) з такими діями:
- Відкрити трек — відкриває повний трек, пов’язаний із вибраним сегментом.
- Приховати трек / Показати трек — керує тим, чи відображається трек на карті.
- Виключити — видаляє сегмент із поточних результатів аналізу.

![Аналізатор треків](@site/static/img/web/web_analyzer_menu.png)

### Графіки {#graphs}

Під картою аналізатор відображає графік, який візуалізує вибрані сегменти. Графік представляє дані лише для сегментів, знайдених між вибраними точками, а не для повних треків.

Графік підтримує перемикання між різними типами даних:
- Висота.
- Нахил.
- Швидкість.
Лише один тип даних відображається одночасно, і зміна його негайно оновлює графік.

Коли присутні кілька сегментів, графік показує дані для кількох сегментів одночасно. Селектор над графіком дозволяє вибрати, скільки графіків сегментів відображається одночасно, та перемикатися між ними.
