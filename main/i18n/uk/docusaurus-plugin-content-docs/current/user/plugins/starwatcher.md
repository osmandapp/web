## Огляд {#overview}

Плагін Star Watcher відображає накладку зоряного неба на карті зі зірками, сузір’ями, Сонцем, Місяцем і планетами. На карті показуються позиції та траєкторії Сонця, Місяця, планет і основних зірок.

## Необхідні параметри налаштування {#required-setup-parameters}
  
Для відображення накладки Star Watcher потрібні такі налаштування:

1. Увімкніть плагін [**Star Watcher**](../plugins/index.md#enable--disable) у розділі *Плагіни* головного меню
2. Використовуйте **Меню → Зоряна карта**, щоб відкрити спеціальний екран із зоряним небом, налаштуваннями та елементами керування часом
3. Виберіть **час і дату** за допомогою слайдерів або кнопок на екрані Зоряної карти
4. Налаштуйте **видимість і прозорість** у налаштуваннях меню "Зоряна карта".

Плагін працює з обома рушіями рендерингу карти, але найкраще функціонує в режимі OpenGL.

## Екран Зоряної карти

**Перейти до:** *Увімкнений плагін → Меню → Зоряна карта* 

![Star map screen](@site/static/img/plugins/starwatcher/view.png)

Спеціальний екран **Зоряної карти** відображає інтерактивний небесний купол із зірками, сузір’ями, планетами, траєкторіями Сонця та Місяця. У нижній частині розташовано панель інструментів із слайдерами часу/дати, кнопками днів і швидкими перемикачами для шарів, таких як лінія горизонту чи мітки.

- Екран відображає повну півсферу неба над вашим розташуванням, узгоджену з напрямком компаса
- Торкніться небесних об’єктів, щоб переглянути деталі, такі як магнітуда, час сходу/заходу чи траєкторії

## Інформація про небесні об’єкти

Торкніться будь-якої **зірки, планети, сузір’я або Сонця/Місяця** на екрані **Зоряної карти** або накладці карти, щоб переглянути детальну інформацію.

![Object info popup](@site/static/img/plugins/starwatcher/object-info.png)

**Відображувані дані:**
- **Азимут**: Напрямок від Півночі (0°-360°), де об’єкт з’являється на небі
- **Висота**: Висота над горизонтом (0° на горизонті, 90° у зеніті)
- **Магнітуда**: Шкала яскравості (-26 для Сонця до +6 для тьмяних зірок; нижче = яскравіше)
- **Час сходу/заходу**: Коли об’єкт сходить над/заходить за горизонт
- **Відстань** (планети): Середня відстань від Землі в а. о./км

**Інтеграція з Вікіпедією**: Торкніться **"Wikipedia"** у спливаючому вікні з інформацією, щоб відкрити сторінку об’єкта в браузері (наприклад, Sirius, Orion, Venus). Працює офлайн для кешованих даних, онлайн для повних статей.

**Довге натискання** на небесні об’єкти дозволяє **закріпити** їх як мітки на карті з оновленням позиції в реальному часі або **поділитися** координатами для групового спостереження за зірками.

Ця функція допомагає ідентифікувати об’єкти на реальному небі, планувати спостереження та вивчати астрономічні факти безпосередньо в OsmAnd.

## AR-пошук зірок (режим камери)

**Перейти до:** *Увімкнений плагін → Меню → Зоряна карта → Кнопка камери* 

Шар **Star Watcher** працює з **камерою пристрою**, щоб увімкнути **доповнену реальність (AR) для спостереження за зірками**. Спрямуйте камеру телефону на реальне нічне небо і побачте зірки, планети, сузір’я, накладені на Сонце/Місяць у реальному часі.

**Як працює AR-пошук зірок:**
- **Живий вид камери** показує реальне небо з прозорими астрономічними накладками, узгодженими з горизонтом/компасом
- **Рухайте камеру**, щоб сканувати небо — об’єкти виділяються, коли з’являються у вашому полі зору
- **Торкайтеся виділених об’єктів**, щоб побачити азимут, висоту, магнітуду, час сходу/заходу та посилання на Вікіпедію
- **Калібрування компаса** необхідне для точного вирівнювання (помахайте телефоном у формі вісімки, якщо потрібно)

**Ідеально для:**
- Ідентифікації тьмяних зірок/планет, невидимих неозброєним оком
- Визначення сузір’їв шляхом переміщення телефону по небу
- Навігації по небу в реальному часі під час походів чи кемпінгу

## Налаштування Star Watcher

*Головне меню → Плагіни → Зоряна карта → Кнопка ⚙️*

Виберіть видимі шари та об’єкти

### Шари зірок

Усі астрономічні дані відображаються як накладки на карті, видимі на масштабах 5-15. Шари проектують небесну сферу на пласку карту.

| Шар | Опис |
|-----|------|
| Зірки | Яскраві зірки (до магнітуди 4-6) з мітками та лініями сузір’їв |
| Сузір’я | З’єднує зірки в знайомі візерунки, такі як Оріон чи Велика Ведмедиця |
| Планети | Позиції та добові траєкторії для Меркурія-Венери-Марса-Юпітера-Сатурна (кольорові іконки) |
| Сонце та Місяць | Дуги, що показують час сходу/заходу та фазу освітлення |
| Горизонт | Лінія, що відокремлює видиму частину неба від землі, з основними напрямками |
