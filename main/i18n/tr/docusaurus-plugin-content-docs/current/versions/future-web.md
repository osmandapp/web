# Web 1.03 {#web-103}

Web 1.03, OsmAnd'ın yeni kullanışlı özellikler ve iyileştirmeler içeren en son web sürümüdür. Bu makale, [Web Planlayıcı](../user/web/index.md) üzerindeki tüm ana güncellemeleri açıklamaktadır.


## Yenilikler {#whats-new}

- Navigasyon UI yeniden tasarımı
- Akıllı klasör desteği
- İz görünüm ayarları (renk & genişlik)
- Toplu taşıma durakları
- POI bağlam menüsü iyileştirmeleri


- [İsteğe bağlı güncellemeler](#optional-updates). Sürüm için ek güncellemeler ve iyileştirmeler.
