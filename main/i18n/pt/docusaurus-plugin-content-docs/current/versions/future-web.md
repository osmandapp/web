# Web 1.03 {#web-103}

Web 1.03 é a versão web mais recente do OsmAnd, que inclui novos recursos e melhorias úteis. Este artigo descreve todas as principais atualizações do [Planejador Web](../user/web/index.md).


## O que há de novo {#whats-new}

- Redesenho da UI de navegação
- Suporte a pastas inteligentes
- Configurações de aparência de trilha (cor e largura)
- Paradas de transporte público
- Melhorias no menu de contexto de POI


- [Atualizações opcionais](#optional-updates). Atualizações e refinamentos adicionais para o lançamento.


## Atualizações opcionais {#optional-updates}

- [Exportar GPX como trilha simplificada](https://github.com/osmandapp/web/issues/436)
- [Horários de funcionamento exibidos no menu de contexto de POI](https://github.com/osmandapp/web/issues/1277)
- [Pastas de favoritos fixadas](https://github.com/osmandapp/web/issues/1193)
- [Melhorias na metadata do Wikimedia](https://github.com/osmandapp/OsmAnd-Issues/issues/2959)
- [Comportamento aprimorado do menu de camadas](https://github.com/osmandapp/web/issues/918)
- [Tooltip de trilha no hover](https://github.com/osmandapp/web/issues/1305)
- [Melhorias na busca para dados pessoais](https://github.com/osmandapp/web/issues/1065)
- [Melhorias na galeria de POI](https://github.com/osmandapp/OsmAnd/issues/22671)
- [Melhorias na seleção de marcadores](https://github.com/osmandapp/web/issues/1297)
- [Comportamento aprimorado da busca por categoria de POI](https://github.com/osmandapp/web/issues/1126)
- [Ocultar compras expiradas na página de conta](https://github.com/osmandapp/web/issues/1328)
