# Comece com o OsmAnd {#begin-with-osmand}

Comece com o OsmAnd (OpenStreetMap Automated Navigation Directions): como instalar e baixar mapas. Tenha acesso a todos os recursos através do *Menu Principal*. Os primeiros passos também se referem à funcionalidade chave do OsmAnd através de uma lista de perguntas simples.

### [Primeiros Passos](./first-steps.md) {#first-steps}

Bem-vindo ao OsmAnd! Explore como baixar mapas, como configurar o aplicativo, como usar o mapa e construir uma rota, como iniciar a navegação e como pesquisar pontos de interesse. Familiarize-se com os pontos chave da política de privacidade do OsmAnd e as permissões necessárias.

### [Baixar Mapas](./download-maps.md) {#download-maps}

Baixar mapas é uma ação necessária para o uso posterior do mapa e navegação no modo offline. Os mapas podem ser baixados tanto navegando pela lista de regiões no *Menu Principal* quanto clicando no território desejado no mapa.

### [Menu Principal](./main-menu.md) {#main-menu}

O *Menu Principal* é uma lista primária de opções disponíveis para uso no aplicativo. Ele fornece acesso rápido à configuração do perfil, configurações globais, pesquisa, dados pessoais, navegação e outros recursos.
