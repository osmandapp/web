## Como encontrar logs de falhas em navegadores da web {#how-to-find-crash-logs-in-web-browsers}

:::note O OsmAnd não coleta nem armazena dados pessoais do usuário
Os logs de falha são gerados pelo seu navegador da web e contêm apenas informações técnicas sobre o problema, como erros do sistema e atividade do navegador no momento da falha. Esses logs não incluem informações pessoais ou dados de rastreamento.
Se você optar por compartilhar logs de falha com o suporte, eles serão usados apenas para diagnosticar e corrigir problemas técnicos.
:::

Para solucionar problemas de forma eficaz com os serviços da web do OsmAnd, é essencial coletar logs de falha do seu navegador da web. Ao relatar problemas com os serviços da web do OsmAnd, fornecer uma captura de tela e um log de falha pode ajudar a diagnosticar e resolver o problema de forma eficiente. Abaixo estão as instruções para acessar os logs de falha em diferentes navegadores da web.

**Google Chrome**:

1. Abra o Chrome e reproduza o problema.
2. Pressione **Command + Option + J** (Mac) ou **Control + Shift + J** (Windows, Linux, ChromeOS) para abrir o **Console JavaScript**.
3. Procure mensagens de erro marcadas em **vermelho**, que podem indicar a causa do problema.
4. Clique com o botão direito do mouse em qualquer lugar do console, selecione **Salvar como…** e salve o arquivo de log.
5. Anexe o arquivo salvo ou copie e cole as mensagens de erro ao relatar o problema.

**Mozilla Firefox**:

1. Abra o Firefox e navegue até a página afetada.
2. Clique em ***Ferramentas → Ferramentas do Navegador → Ferramentas do Desenvolvedor Web → Console***, ou pressione **Ctrl + Shift + I** (Windows/Linux) ou **Cmd + Opt + I** (macOS) para abrir o **Console do Desenvolvedor**.
3. Localize quaisquer mensagens de erro ou avisos relacionados ao problema.
4. Para exportar logs, clique com o botão direito do mouse dentro do console, selecione **Exportar Saída do Console** e salve o arquivo.
5. Compartilhe o arquivo de log salvo junto com uma descrição do problema.

**Microsoft Edge**:

1. Abra o Edge e tente reproduzir o erro.
2. Pressione **F12** ou **Ctrl + Shift + I** para abrir as **Ferramentas do Desenvolvedor**.
3. Clique na guia **Console** para visualizar quaisquer erros ou avisos.
4. Se necessário, clique com o botão direito do mouse dentro do console, escolha **Salvar como…** e salve o log de erro para envio.

**Dicas de solução de problemas**:

- Se o site travar completamente, tente atualizar a página e verificar os logs novamente.
- Certifique-se de estar usando a **versão mais recente do seu navegador**, pois versões desatualizadas podem causar problemas inesperados.
- Desative quaisquer extensões do navegador que possam interferir na funcionalidade do site e teste novamente.
- Se o problema persistir, reinicie o navegador ou tente usar um diferente.
