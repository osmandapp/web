## Visão geral {#overview}

Na web, os usuários podem acessar, gerenciar e criar seus dados pessoais, como Trilhas e Favoritos, diretamente na web.

- Trilhas: Os usuários podem carregar, visualizar, editar ou criar novas trilhas online. Todas as alterações são sincronizadas automaticamente com o OsmAnd Cloud, garantindo que os dados estejam disponíveis em todos os dispositivos conectados.
- Favoritos: Crie e organize locais salvos na web, com sincronização automática para o seu aplicativo OsmAnd para acesso rápido e navegação.

A integração com o OsmAnd Cloud permite uma sincronização perfeita, permitindo que os usuários gerenciem seus dados convenientemente de qualquer dispositivo.

## Trilhas na web {#tracks-on-the-web}

É um recurso pago <ProFeature/>. Primeiro, você precisa fazer login no seu OsmAnd Pro:

![Login de trilha](@site/static/img/web/track_loggin.png)

Para a seção de Trilhas:

- Mostrar trilhas do [OsmAnd Cloud](#cloud-tracks).
- Adicionar trilhas no mapa (pasta **Visível no mapa**).
- Visualizar todas as informações e gráficos das trilhas.
- Modificar trilhas e adicioná-las à Nuvem.
- Baixar e excluir trilhas.
- Criar novas pastas ou excluí-las.
- Baixar pastas como coleção OSF ou OBF.

### Visível no mapa {#visible-on-the-map}

Aqui está uma lista de trilhas visíveis no mapa.

_Ative/desative_ as trilhas para mostrá-las ou não no mapa.

![Visível no mapa](@site/static/img/web/visible.png)

### Menu da pasta de trilhas {#track-folder-menu}

Clique no botão de *Três pontos* **⋮** para abrir o menu *Pasta de Trilhas*. Aqui você pode baixar todas as trilhas da pasta escolhida como:

- Baixar como OSF.
- Baixar como Coleção OBF:
  Você pode obter o mapa OBF a partir desses dados: [arquivo OBF](https://osmand.net/docs/technical/osmand-file-formats/osmand-obf/) ou [Viagem OBF](https://osmand.net/blog/routes#generated-travel-routes).

  - [arquivo OBF](https://osmand.net/docs/technical/osmand-file-formats/osmand-obf/):

  Você pode baixar o mapa offline OBF (OsmAnd Binary Format) e abri-lo com o OsmAnd no dispositivo, é adequado para exibir uma grande quantidade de trilhas que serão visíveis no mapa.

  - [Viagem OBF](https://osmand.net/blog/routes#generated-travel-routes):

  Você também pode importar um mapa de trilhas como um livro de viagens, o que permitirá selecionar uma trilha específica no mapa e usá-la como um GPX normal. O livro de viagens também suporta recursos como exibir trilhas como pontos, filtrar trilhas por tipo de atividade e filtrar pontos de passagem.

- _Renomear_ a pasta escolhida.
- _Excluir_ a pasta escolhida.

![Menu da pasta de trilhas](@site/static/img/web/collection.png)

### Trilhas da Nuvem {#cloud-tracks}

As trilhas GPX que você tem no [OsmAnd Cloud](../personal/osmand-cloud.md) estarão disponíveis para exibição e edição após o login. Apenas para **usuários Pro** <ProFeature/> e para usuários [**OsmAnd Start**](../personal/osmand-cloud.md#osmand-start) (que podem baixar seus dados mesmo após o término da assinatura Pro).

Os seguintes recursos estão disponíveis após a escolha da trilha da nuvem:

- Telas de informação:
  - *Informações* - exibindo dados da trilha.
  - *Elevação* - gráfico de elevação.
  - *Velocidade* - gráfico de velocidade.
  - *Inclinação* - gráfico de inclinação.
  - *Configurações* - lista de perfis de navegação para a ferramenta *Criar rota*.

- Botões de ação:
  - *Editar trilha* - adiciona a trilha da nuvem ao local para edição (botão _Editar localidade_ no painel).
  - *Baixar GPX* - baixa a trilha para o PC.
  - *Recalcular Elevação* (Satélite) - exibe o gráfico de elevação da trilha selecionada.
  - *Excluir trilha* - exclui a trilha.
  - *Fechar trilha* - fecha a trilha.

![Edição de GPX da nuvem do OsmAnd Web](@site/static/img/web/cloud_track.png)

## Favoritos na web {#favorites-on-the-web}

Após registrar um [**OsmAnd Pro**](../personal/osmand-cloud.md#login) e para [**OsmAnd Start**](../personal/osmand-cloud.md#osmand-start), seus Favoritos são exibidos no menu Favoritos.
As seguintes ações estão disponíveis:

- *Mostrar no mapa* - mostra os pontos favoritos da pasta escolhida no mapa.
- *Renomear* - nome e descrição da pasta de favoritos.
- *Baixar* - baixa a pasta de favoritos escolhida.
- *Excluir* - exclui a pasta de favoritos escolhida.

![Edição de Favoritos da nuvem do OsmAnd Web](@site/static/img/web/favorites.png)

### Adicionar / Editar Favorito {#add--edit-favorite}

[Para adicionar](../personal/favorites.md#manage-favorites) um novo ponto favorito, clique com o botão direito do mouse na tela. Para iniciar a edição, clique no ponto favorito, a tela do menu de edição é a mesma do [menu no aplicativo OsmAnd](../personal/favorites.md#create).

![Adicionar Favoritos do OsmAnd Web](@site/static/img/web/web_favorites_add.png)
