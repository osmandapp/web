### [Introdução](./web-overview.md) {#introduction}

Nesta seção, você encontrará uma descrição do nosso [Web Planner](https://osmand.net/map).

### [Conta](./web-cloud.md) {#account}

Esta seção explica como fazer login e se registrar em [www.osmand.net/map](https://osmand.net/map/) e destaca os benefícios que os usuários recebem após o registro.

### [Mapa](./web-map.md) {#map}

Esta seção aborda a interface do mapa web do OsmAnd, incluindo o menu do mapa, configuração de camadas, configurações e links de URL.

### [Clima](./web-weather.md) {#weather}

Esta seção explica como visualizar condições climáticas e previsões no mapa web, incluindo previsões baseadas em tempo e sobreposições climáticas.

### [Trilhas](./web-tracks.md) {#tracks}

Esta seção aborda o trabalho com trilhas GPX pessoais na web, incluindo visualização, edição, organização e sincronização com o OsmAnd Cloud.

### [Favoritos](./web-favorites.md) {#favorites}

Lugares salvos exibidos no mapa, com opções para organizá-los em pastas, gerenciar acesso e reutilizá-los para navegação ou planejamento de rotas.

### [Analisador de Trilhas](./web-tracks-analyzer.md) {#tracks-analyzer}

Esta seção foca na análise de segmentos de trilhas entre pontos selecionados no mapa e na comparação de parâmetros chave entre múltiplas trilhas.

### [Navegação](./web-navigation.md) {#navigation}

Esta seção explica como construir e visualizar rotas de navegação no mapa web, incluindo perfis de roteamento, configurações e detalhes da rota.

### [Planejar Rota](./planner.md) {#plan-route}

Esta seção fornece informações sobre planejamento de rotas ou criação de navegação usando diferentes perfis.

### [Pesquisar](./web-search.md) {#search}

Esta seção explica como encontrar pontos de interesse (POIs) de diferentes categorias no mapa, realizar uma pesquisa clássica e exibir ícones de categoria no mapa.
