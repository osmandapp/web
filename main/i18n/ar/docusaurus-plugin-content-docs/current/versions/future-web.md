# الويب 1.03 {#web-103}

الويب 1.03 هو أحدث إصدار ويب من OsmAnd، والذي يتضمن ميزات وتحسينات جديدة مفيدة. تصف هذه المقالة جميع التحديثات الرئيسية [لمخطط الويب](../user/web/index.md).


## ما الجديد {#whats-new}

- إعادة تصميم واجهة المستخدم للملاحة
- دعم المجلدات الذكية
- إعدادات مظهر المسار (اللون والعرض)
- محطات النقل العام
- تحسينات قائمة السياق لـ POI


- [تحديثات اختيارية](#optional-updates). تحديثات وتحسينات إضافية للإصدار.


## تحديثات اختيارية {#optional-updates}

- [تصدير GPX كمسار مبسط](https://github.com/osmandapp/web/issues/436)
- [ساعات العمل المعروضة في قائمة السياق لـ POI](https://github.com/osmandapp/web/issues/1277)
- [مجلدات المفضلة المثبتة](https://github.com/osmandapp/web/issues/1193)
- [تحسينات بيانات وصفية ويكيميديا](https://github.com/osmandapp/OsmAnd-Issues/issues/2959)
- [تحسين سلوك قائمة الطبقات](https://github.com/osmandapp/web/issues/918)
- [تلميح مسار عند التمرير](https://github.com/osmandapp/web/issues/1305)
- [تحسينات البحث عن البيانات الشخصية](https://github.com/osmandapp/web/issues/1065)
- [تحسينات معرض POI](https://github.com/osmandapp/OsmAnd/issues/22671)
- [تحسينات اختيار العلامات](https://github.com/osmandapp/web/issues/1297)
- [تحسين سلوك البحث عن فئة POI](https://github.com/osmandapp/web/issues/1126)
