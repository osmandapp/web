## جميع الميزات والإمكانيات {#all-features-and-capabilities}

OsmAnd هو **تطبيق قوي للملاحة ورسم الخرائط** يقدم مجموعة واسعة من الميزات عبر الإنترنت وغير المتصلة بالإنترنت. توفر هذه الصفحة نظرة عامة مفصلة على جميع الميزات المتاحة في الإصدارات المختلفة من التطبيق.



## الملاحة {#navigation}

ميزات الملاحة الأساسية:

- الملاحة خطوة بخطوة مع توجيهات صوتية
- إعادة توجيه تلقائية عند الانحراف عن المسار
- الوقت المقدر للوصول (ETA)
- دعم نقاط الطريق الوسيطة
- إرشاد المسار
- تحذيرات تجاوز السرعة
- تعديلات المسار مع مراعاة حركة المرور

أوضاع النقل المدعومة:

- سيارة
- دراجة
- مشاة
- النقل العام
- الملاحة البحرية
- المشي لمسافات طويلة

الجدول:

| الميزة | OsmAnd المجاني | OsmAnd+ (خرائط+) | OsmAnd Pro |
|---------|------------|----------------|------------|
| الملاحة خطوة بخطوة | ✔ | ✔ | ✔ |
| إرشاد المسار | ✔ | ✔ | ✔ |
| نقاط الطريق الوسيطة | ✔ | ✔ | ✔ |
| إعادة التوجيه مع مراعاة حركة المرور | - | - | ✔ |
| Android Auto / CarPlay | - | ✔ | ✔ |


## الخرائط والعرض {#maps-and-rendering}

الخرائط والتحديثات غير المتصلة بالإنترنت:

- خرائط إقليمية وقطرية قابلة للتنزيل
- تحديثات شهرية للخرائط
- تحديثات كل ساعة عبر OsmAnd Live

أنواع الخرائط المدعومة:

- خريطة OpenStreetMap القياسية
- خرائط الأقمار الصناعية (عبر الإنترنت)
- خرائط التضاريس ثلاثية الأبعاد
- خطوط الكنتور (الطبوغرافيا)
- الأعماق الملاحية والهيدرولوجيا

طبقات إضافية:

- سطح الطريق وجودته
- مسارات المشي ومسارات الدراجات
- طبقات OpenStreetMap غير المتصلة بالإنترنت
- طبقات الطقس الديناميكية

الجدول:

| الميزة | OsmAnd المجاني | OsmAnd+ (خرائط+) | OsmAnd Pro |
|---------|------------|----------------|------------|
| خرائط غير متصلة بالإنترنت | ✔ | ✔ | ✔ |
| تحديثات شهرية للخرائط | ✔ | ✔ | ✔ |
| تحديثات كل ساعة (OsmAnd Live) | - | - | ✔ |
| التضاريس ثلاثية الأبعاد | - | - | ✔ |
