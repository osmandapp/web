## What's new


- Redesigned "My Places" — with Favorites subfolders, pinned folders, and quicker actions
- Rebuilt "Plan a Route" — add route and POI points, and combine different route types
- New Astronomy plugin: star map with a detailed database for each object, visibility graphs, and an observation schedule
- Added "Terrain shadows" visualization type
- Added support for Road-only maps
- Edit the color palette for Tracks and Terrain
- Auto-organize tracks by year, length, activity, and more
- Split multi-track GPX files into separate files on import
- Set a default appearance for Track folders
- Added a bicycle width setting to avoid narrow cycleways
- Added an OBD-II adapter voltage widget
- Added a "Pause spoken audio" option to pause podcasts or audiobooks during navigation instructions
- Full icon selection now available for profiles
- Improved performance when managing a large number of Favorites
- Fixed an issue with the CarPlay trial period
- [Bug fixes](#bug-fixes).
