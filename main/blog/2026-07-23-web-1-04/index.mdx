OsmAnd Web 1.04 — Now Available!

OsmAnd Web 1.04 introduces a range of improvements to search, track management, and map interaction. This release adds support for finding Favorites and tracks by name, enhances spatial search and search results, and makes importing GPX tracks as easy as dragging them onto the map. It also brings Travel track filters by activity and year, Live Location Sharing, improved map interaction with Zoom-to-Fit and Focus mode, faster context menus for map objects, and new subscription paywalls for premium Web features.

Explore the latest improvements and enjoy an even more powerful OsmAnd Web experience.

<!--truncate-->

## What's new

- [Search Favorites and tracks by name](https://github.com/osmandapp/web/issues/1065);
- [Zoom-to-Fit and Focus mode for Favorites and tracks](https://github.com/osmandapp/web/issues/1445);
- [Drag-and-drop GPX track import](https://github.com/osmandapp/web/issues/1283);
- [Travel tracks now support activity and year filters](https://github.com/osmandapp/OsmAnd-Issues/issues/2708);
- [Live Location Sharing](https://github.com/osmandapp/OsmAnd-Issues/issues/3153);
- [Context menus now include a Close button](https://github.com/osmandapp/web/issues/1867);
- [Improved context menus for map objects](https://github.com/osmandapp/OsmAnd-Issues/issues/3263);
- [Enhanced spatial search with autocomplete](https://github.com/osmandapp/web/issues/1893);
- [Subscription paywalls for premium Web features](https://github.com/osmandapp/OsmAnd-Issues/issues/3029);
- Bug fixes.
